FLISoL 2017 Blumenau

Install Fest

É um espaço destinado para que a comunidade instale o sistema operacional Linux e aplicativos de código aberto no Windows. Durante o Install Fest você terá o acompanhamendo de uma pessoa com conhecimento técnico no assunto.

A quem de destina?

A todos que desejam usar software livre de alguma forma. Você provalmente gostará de participar se deseja:

  • Usar Linux (Ubuntu, Fedora, entre outras distribuições).
  • Resolver algum problema com o seu Linux.
  • Instalar um aplicativo de código aberto no Windows (LibreOffice, Firefox, entre outros).

Regras

Traga preferencialmente o seu notebook, desktops poderão ser um problema devido ao espaço limitado. Estaremos nas salas de aula da FURB.

Traga carregadores, adaptadores para o padrão novo e se possível extensões.

Para participar do evento você deverá obrigatóriamente assinará um termo de resposabilidade onde concordará que todo e qualquer problema decorrente do sistema operacional ou dos aplicativos agregados não é responsabilidade dos organizadores do FLISoL e/ou dos patrocinadores e apoiadores.

Você deve fazer backup dos seus dados previamente, o evento não irá se responsabilizar por backups durante a instalação do Linux.

Não nos resposabilizaremos pelo transporte do seu equipamento.

Não nos resposabilizaremos por furtos e roubos de equipamentos. Lembre-se, você estará em um ambiente público, tome cuidado com o seu equipamento.
